## Installation

1. First, clone the repository:

```
git clone https://github.com/Phoudthavisack/nodejs-hack-wifi.git

```

1. Install the dependencies:

```
cd nodejs-hack-wifi
npm install

```

## Starting the application

1. Start the NodeJS:

```
node index.js

```
